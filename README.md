# ft_containers

This project is an implementation of the containers of the C++ standard library. The containers are implemented in C++98.

## Brief

The containers are implemented in the following order:

- [x] Vector
- [x] Map
- [x] Stack
- [ ] Queue
- [ ] Double ended queue
- [ ] Set
- [ ] Multiset
- [ ] Multimap
